# 模板驱动图片任务

本上下文描述 Agent 根据提示词模板收集输入、构建完整提示词并执行图片任务时使用的核心语言。

## Language

**模型配置**:
设备使用者为应用提供的模型服务连接信息，分为图片模型配置和文本模型配置，两者可以共用或分别设置。配置中的 API 凭据仅保存在设备安全存储中，不进入备份或导出文件。
_Avoid_: API 设置、供应商配置、密钥配置

**提炼会话**:
用户从提交外部完整提示词开始，到确认图鉴条目方案或放弃为止的一次模板提炼过程。外部完整提示词仅在本机处理，完成或放弃后不再保留。
_Avoid_: 提炼任务、模板导入任务

**提示词图鉴**:
由 Imagemon Promptdex 管理、可持续扩充的提示词模板集合。每个有效模板都是一条独立的图鉴条目。
_Avoid_: 模板库、模板索引

**图鉴条目**:
提示词图鉴中的一个有效提示词模板，可被独立选择并用于一次图片任务。
_Avoid_: 模板记录、模板配置

**条目展示信息**:
用于浏览、搜索和识别图鉴条目的标题、用途说明、分类、搜索标签、封面示例图和任务类型标识，不参与完整提示词构建。个人图鉴条目的展示标题、分类、搜索标签和封面可调整，用途说明与任务类型随执行模板保持不可变；封面从该条目的成功图片结果派生并长期保留至条目删除，不受图片结果保留期影响。
_Avoid_: 模板元数据、展示元数据、模板介绍

**图鉴分类**:
产品维护、用于统一浏览内置与个人图鉴条目的单选分类。首版分类为信息图表、社交媒体、营销广告、电商商品、品牌设计、摄影写真、插画艺术、UI 与产品设计、图片编辑和其他。
_Avoid_: 模板类型、标签、目录

**可用图鉴条目**:
当前允许创建新图片任务的图鉴条目。个人图鉴条目被删除后不再可用，关联任务历史仍可查看但不能再次生成。
_Avoid_: 有效模板、启用模板、在线模板

**内置图鉴条目**:
随产品提供、所有用户均可使用且用户不可修改的图鉴条目。
_Avoid_: 官方模板、系统模板、公共模板

**个人图鉴条目**:
仅保存在当前设备、由设备使用者管理的不可变图鉴条目。条目长期保留至使用者主动删除，可通过重新提炼创建替代条目，但不能直接修改已有条目。
_Avoid_: 私有模板、用户模板、自定义模板

**图鉴条目导出**:
用户对个人图鉴条目进行完整备份和迁移的结构化文件，包含提示词模板正文，但不包含任务完整提示词、图片结果或内部执行信息。内置图鉴条目不可导出，导出文件再次引入时仍须经过模板提炼与确认。
_Avoid_: 模板下载、模板分享、模板备份

**图片规格**:
图片任务的画面方向、尺寸、质量、格式和数量等交付属性。用户未指定时采用提示词图鉴约定的默认规格。
_Avoid_: 内容要求、设计要求

**提示词模板**:
声明图片任务所需输入并提供提示词正文的可复用文档。模板自身不包含某次任务的具体用户内容。
_Avoid_: 完整提示词、用户提示词

**完整提示词**:
由提示词模板正文和某次任务的具体输入共同组成，能够直接用于图片任务的最终指令。
_Avoid_: 用户提示词、原始提示词

**模板提炼**:
以计划用途界定目标，从外部完整提示词中识别并保留与目标兼容的可复用设计知识，形成新的图鉴条目，同时移除仅属于原始任务的具体内容。
_Avoid_: 提示词转换、提示词改写

**提炼补充**:
外部完整提示词未提供、由 Agent 为完善图鉴条目而提出的新规则；任何提炼补充在写入图鉴条目前都必须获得用户批准。
_Avoid_: 自动完善、默认规则

**正文结构**:
提示词模板正文中为表达可复用规则而采用的自由组织方式，不要求固定章节名称或数量。
_Avoid_: 模板规范、固定章节

**模板输入**:
图鉴条目为每次使用时会变化的任务内容声明的最小输入集合；定义模板身份的规则属于正文，图片规格属于图片规格。
_Avoid_: 模板参数、可配置项

**图片任务**:
一次根据完整提示词生成新图片或编辑已有图片的请求，使用设备使用者配置的 API 凭据直接调用模型服务，首版每次只产生一张图片，且同一时间只允许一个模型调用。使用者可以取消本地等待，但不能保证模型服务停止处理或避免费用。
_Avoid_: 生图流程、绘图请求、生成任务

**图片结果**:
图片任务产生并保存在当前设备上的图片，长期保留至设备使用者主动删除。
_Avoid_: 生成结果、作品、成品

**任务历史**:
保存在当前设备上的图片任务历史记录，长期保留至设备使用者主动删除。再次生成使用原任务的图鉴条目版本，并要求使用者确认或修改回填内容后创建新图片任务；个人图鉴条目被删除后，关联记录仍可查看但不能再次生成。
_Avoid_: 生成历史、作品历史

**生成任务**:
不使用已有图片作为输入、根据完整提示词创建新图片的图片任务。
_Avoid_: 图片任务、编辑任务

**编辑任务**:
使用一张已有图片作为输入、根据完整提示词修改图片的图片任务。
_Avoid_: 图片任务、生成任务
