# 模板驱动图片任务

本上下文描述 Agent 根据提示词模板收集输入、构建完整提示词并执行图片任务时使用的核心语言。

## Language

**设备使用者**:
在当前设备上管理提示词图鉴、模型配置、图片结果、任务历史和备份恢复的人；不等同于执行 Skill 的 Agent 或维护项目的开发者。
_Avoid_: 用户、Agent、开发者

**模型配置**:
设备使用者为应用提供的 OpenAI 兼容模型服务连接信息，分为图片模型配置和文本模型配置，两者可以共用或分别设置，并可分别保留多个配置与默认配置。API 凭据仅保存在设备安全存储中，不进入备份或导出文件。
_Avoid_: API 设置、供应商配置、密钥配置

**就绪模型配置**:
已具备模型调用所需非敏感连接信息和安全存储 API 凭据，最近一次相关模型配置测试结果为通过，且影响连接或调用行为的字段及 API 凭据在该次测试通过后未变化的模型配置，可被设为默认配置并用于图片任务或模板提炼。
_Avoid_: 有效配置、完整配置、可用配置

**默认模型配置**:
设备使用者为图片模型或文本模型选择的首选就绪模型配置；同一模型类型最多有一个默认模型配置。
_Avoid_: 默认 API、默认供应商、当前模型

**首次设置**:
手机端首次运行时帮助设备使用者创建或跳过默认模型配置的功能性设置流程；完成或跳过后不再强制出现，不等同于营销式 onboarding。
_Avoid_: 新手引导、欢迎页、初始化向导

**模型配置快照**:
图片任务执行时保存的非敏感模型配置记录，用于说明当时使用的配置名称、模型服务、模型名称和调用选项，不包含 API 凭据或安全存储引用。
_Avoid_: 配置引用、密钥快照、供应商记录

**模型调用**:
应用使用设备使用者配置的模型服务执行图片任务、模板提炼、模型配置测试或拉取模型列表的外部请求；同一时间全应用只允许一个模型调用。按用途分为业务模型调用和诊断类模型调用。
_Avoid_: API 请求、后台任务、模型队列

**业务模型调用**:
直接产出设备使用者可保留资产的模型调用，包括图片任务和模板提炼。完成、失败或结果不确定时产生业务调用提示，等待期间显示 loading，首版不设固定应用侧超时。
_Avoid_: 前台任务、用户调用、生成请求

**诊断类模型调用**:
仅用于验证或辅助填写模型配置、不产出可保留资产的模型调用，包括模型配置测试连接和拉取模型列表。使用 30 秒应用侧超时，不产生业务调用提示。
_Avoid_: 辅助调用、探测请求、系统调用

**全局模型调用状态**:
全局模型调用锁被占用期间展示的应用级状态，说明当前进行中的模型调用类型（图片任务、模板提炼、模型配置测试连接或拉取模型列表），并提供返回当前调用页面的入口；覆盖所有模型调用类型，不显示用量或费用。区别于仅服务业务模型调用的业务调用提示。
_Avoid_: 后台任务指示、进度条、未读提示

**模型配置测试连接**:
设备使用者在模型配置详情页显式触发、用于验证当前模型配置能否建立基本连接的诊断性模型调用；结果只影响模型配置状态，不进入任务历史。
_Avoid_: 自动测试、连通性探测、试运行

**提示词图鉴**:
由 Imagemon Promptdex 管理、可持续扩充的提示词模板集合。每个有效模板都是一条独立的图鉴条目。
_Avoid_: 模板库、模板索引

**图鉴条目**:
提示词图鉴中的一个有效提示词模板，以唯一名称识别，可被独立选择并用于一次图片任务。
_Avoid_: 模板记录、模板配置

**内置图鉴条目**:
随应用版本提供、对设备使用者只读，且不能被个人图鉴条目覆盖的图鉴条目。
_Avoid_: 官方模板、系统模板、公共模板

**个人图鉴条目**:
仅保存在当前设备、由设备使用者管理的图鉴条目。条目长期保留至使用者主动删除，可编辑条目展示信息，但不能直接修改执行语义。
_Avoid_: 私有模板、用户模板、自定义模板

**条目展示信息**:
用于浏览、搜索和识别图鉴条目的标题、用途说明、分类、搜索标签、可选封面示例图，以及由输入声明推断出的任务类型标识，不参与完整提示词构建。
_Avoid_: 模板元数据、展示元数据、模板介绍

**图鉴分类**:
条目展示信息中用于分组和筛选图鉴条目的单个非空文本分类；内置图鉴条目使用应用约定分类，个人图鉴条目可以使用自由分类。
_Avoid_: 分类枚举、多级分类、目录

**搜索标签**:
条目展示信息中用于搜索和识别图鉴条目的若干短文本标签，不参与完整提示词构建。
_Avoid_: 关键词、标签分类、话题

**Promptdex Markdown**:
用于导入和导出个人图鉴条目的 Markdown 文件格式，包含输入声明、提示词模板正文和条目展示信息；不等同于 Skill 内置模板文件。
_Avoid_: 普通 Markdown、模板文件、完整提示词文件

**图鉴条目导出**:
设备使用者对个人图鉴条目进行完整备份和迁移的 Promptdex Markdown 文件，包含输入声明、提示词模板正文和除封面示例图文件以外的条目展示信息；不包含任务完整提示词、任务历史、任务快照、图片结果或 API 凭据。符合当前契约的文件经本地校验后可直接导入。
_Avoid_: 模板下载、模板分享、模板备份

**ZIP 备份**:
设备使用者主动导出的应用级迁移文件，包含个人图鉴条目、任务历史、任务快照、图片结果、非敏感模型配置、应用默认规格和内部附件，不包含 API 凭据、提炼草稿或模型配置测试状态。
_Avoid_: 自动同步、云备份、系统备份

**备份格式版本**:
ZIP 备份 manifest 中声明的备份文件结构版本，用于判断当前应用能否恢复该备份；不等同于应用版本。
_Avoid_: 应用版本、数据版本、迁移版本

**只读恢复模式**:
应用升级时本地数据迁移失败回滚后进入的应用状态，期间保留旧数据、允许导出完整 ZIP 备份用于抢救，但禁止创建或修改任何资产，也禁止发起 ZIP 恢复；区别于迁移成功后的正常运行状态和恢复进行中。
_Avoid_: 安全模式、降级模式、恢复进行中

**恢复预检**:
ZIP 备份恢复前的只读检查和预览过程，用于校验备份格式版本、资产清单、哈希和导入冲突，并在设备使用者确认前不写入应用数据。
_Avoid_: 试恢复、模拟导入、恢复草稿

**恢复进行中**:
ZIP 备份恢复确认后到导入完成或失败回滚前的应用状态；期间禁止图片任务、模板提炼、模型配置测试、其他会修改应用数据的操作，以及另一次 ZIP 导出或恢复。
_Avoid_: 后台恢复、恢复任务、恢复队列

**导出进行中**:
ZIP 备份导出确认后到导出完成、失败或取消前的应用状态；期间禁止图片任务、模板提炼、模型配置测试、其他会修改应用数据或读取整库生成备份的操作，以及另一次 ZIP 导出或恢复。ZIP 导出与恢复互为应用级独占迁移操作，二者互斥且各自不可并发自身。
_Avoid_: 后台导出、导出任务、导出队列

**图片规格**:
图片任务的画面方向、尺寸、质量、格式和数量等交付属性。用户未指定时采用应用默认规格，图鉴条目不声明默认图片规格。
_Avoid_: 内容要求、设计要求

**应用默认规格**:
应用用于预填图片任务的全局默认图片规格，可由设备使用者修改，不属于任何图鉴条目。
_Avoid_: 模板默认规格、模型默认规格、推荐规格

**提示词模板**:
声明图片任务所需输入并提供提示词正文的可复用文档。模板自身不包含某次任务的具体用户内容。
_Avoid_: 完整提示词、用户提示词

**完整提示词**:
由提示词模板正文和某次任务的具体输入共同组成，能够直接用于图片任务的最终指令。
_Avoid_: 用户提示词、原始提示词

**模板提炼**:
以计划用途界定目标，从外部完整提示词中识别并保留与目标兼容的可复用设计知识，形成新的图鉴条目，同时移除仅属于原始任务的具体内容。
_Avoid_: 提示词转换、提示词改写

**提炼补充**:
外部完整提示词未提供、由 Agent 为完善图鉴条目而提出的新规则；任何提炼补充在写入图鉴条目前都必须获得用户批准。
_Avoid_: 自动完善、默认规则

**提炼方案**:
模板提炼产生、等待设备使用者确认的拟写入图鉴条目方案，概括模板名称、用途、任务类型、输入集合、保留规则、删除规则和提炼补充；确认后才能写入个人图鉴条目。
_Avoid_: 草稿条目、提炼结果、生成草稿

**提炼草稿**:
模板提炼过程中尚未确认写入的本地状态记录，用于保存未完成或结果不确定的提炼流程，不是图鉴条目。
_Avoid_: 草稿条目、临时模板、提炼历史

**正文结构**:
提示词模板正文中为表达可复用规则而采用的自由组织方式，不要求固定章节名称或数量。
_Avoid_: 模板规范、固定章节

**模板输入**:
图鉴条目为每次使用时会变化的任务内容声明的最小输入集合；定义模板身份的规则属于正文，图片规格属于图片规格。
_Avoid_: 模板参数、可配置项

**图片任务**:
一次根据完整提示词生成新图片或编辑已有图片的请求，使用设备使用者配置的 API 凭据直接调用模型服务。
_Avoid_: 生图流程、绘图请求、生成任务

**图片结果**:
图片任务产生并保存在当前设备上的图片文件，作为独立资产长期保留至设备使用者主动删除，不因任务历史删除而删除。
_Avoid_: 生成结果、作品、成品

**任务历史**:
保存在当前设备上的图片任务历史记录，作为独立资产长期保留至设备使用者主动删除，不因图片结果删除而删除。
_Avoid_: 生成历史、作品历史

**任务状态**:
任务历史用于表达图片任务当前或最终结果的状态，包括进行中、完成、失败和状态未知。
_Avoid_: 进度、执行阶段、运行状态

**业务调用提示**:
设备使用者对图片任务和模板提炼这两类业务模型调用产生的、仅保存在当前设备的布尔可发现性提示，表达存在待查看的成功结果、待确认的提炼方案，或未查看的失败与需要处理项；只表达有无、不显示计数，进入对应详情或处理页后清除，不进入 ZIP 备份，恢复导入不生成。模型配置测试连接和拉取模型列表等诊断类模型调用不产生此提示。
_Avoid_: 未读数、红点计数、通知记录、系统通知

**错误摘要**:
图片任务失败时保存的结构化、非敏感错误说明，用于解释失败原因，不包含完整响应体、请求头、API 凭据或底层堆栈。
_Avoid_: 错误日志、异常详情、失败响应

**模型配置测试失败摘要**:
模型配置测试连接失败时在当前模型配置页面会话内临时显示的结构化、非敏感错误说明，用于解释刚发生的测试失败；离开页面或应用重启后丢失，不进入持久模型配置状态、任务历史或 ZIP 备份。
_Avoid_: 测试日志、连接日志、失败响应

**任务快照**:
图片任务执行时保存的图鉴条目、任务输入、图片规格、模型配置快照和完整提示词记录，用于查看和解释任务历史；其中完整提示词只能只读展示，任务快照不作为再次执行图片任务的输入。
_Avoid_: 历史详情、任务备份、执行缓存

**生成任务**:
不使用已有图片作为输入、根据完整提示词创建新图片的图片任务。
_Avoid_: 图片任务、编辑任务

**编辑任务**:
使用一张已有图片作为输入、根据完整提示词修改图片的图片任务。
_Avoid_: 图片任务、生成任务
