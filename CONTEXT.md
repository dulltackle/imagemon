# 模板驱动图片任务

本上下文描述 Agent 根据提示词模板收集输入、构建完整提示词并执行图片任务时使用的核心语言。

## Language

**提示词图鉴**:
由 Imagemon Promptdex 管理、可持续扩充的提示词模板集合。每个有效模板都是一条独立的图鉴条目。
_Avoid_: 模板库、模板索引

**图鉴条目**:
提示词图鉴中的一个有效提示词模板，可被独立选择并用于一次图片任务。
_Avoid_: 模板记录、模板配置

**图片规格**:
图片任务的画面方向、尺寸、质量、格式和数量等交付属性。用户未指定时采用提示词图鉴约定的默认规格。
_Avoid_: 内容要求、设计要求

**提示词模板**:
声明图片任务所需输入并提供提示词正文的可复用文档。模板自身不包含某次任务的具体用户内容。
_Avoid_: 完整提示词、用户提示词

**完整提示词**:
由提示词模板正文和某次任务的具体输入共同组成，能够直接用于图片任务的最终指令。
_Avoid_: 用户提示词、原始提示词

**模板提炼**:
以计划用途界定目标，从外部完整提示词中识别并保留与目标兼容的可复用设计知识，形成新的图鉴条目，同时移除仅属于原始任务的具体内容。
_Avoid_: 提示词转换、提示词改写

**提炼补充**:
外部完整提示词未提供、由 Agent 为完善图鉴条目而提出的新规则；任何提炼补充在写入图鉴条目前都必须获得用户批准。
_Avoid_: 自动完善、默认规则

**正文结构**:
提示词模板正文中为表达可复用规则而采用的自由组织方式，不要求固定章节名称或数量。
_Avoid_: 模板规范、固定章节

**模板输入**:
图鉴条目为每次使用时会变化的任务内容声明的最小输入集合；定义模板身份的规则属于正文，图片规格属于图片规格。
_Avoid_: 模板参数、可配置项

**图片任务**:
一次根据完整提示词生成新图片或编辑已有图片的请求。信息充分时直接执行；仅当缺口影响任务意图或事实准确性时才向用户追问。
_Avoid_: 生图流程、绘图请求、生成任务

**生成任务**:
不使用已有图片作为输入、根据完整提示词创建新图片的图片任务。
_Avoid_: 图片任务、编辑任务

**编辑任务**:
使用一张已有图片作为输入、根据完整提示词修改图片的图片任务。
_Avoid_: 图片任务、生成任务
