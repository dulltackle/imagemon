# 明亮后印象派手绘重绘

将输入图片重新绘制为一张 luminous post-impressionist illustration。必须保留原图中的主要主体、身份特征、视角、核心构图、空间关系和可识别内容。最终画面不是在原照片上叠加滤镜，而是把原图内容重新诠释为统一的手绘插画。

## 绘画语言

- 使用 post-impressionist 与 plein-air painting 气质，画面应像在自然光中完成的手绘作品。
- 呈现 watercolor-oil hybrid 的混合媒介质感：柔和水彩晕染、透明色层与轻微油画触感并存。
- 笔触松散、流动、有手势感，保留 visible paint texture 和自然的不均匀边缘。
- 允许局部 soft-focus 与轻微 bokeh impression，但不能模糊主体的关键识别特征。
- 避免写实摄影、3D 渲染、硬边矢量、赛璐璐动画、厚重暗调油画和数码喷枪感。

## 色彩与光线

- 整体色彩明亮、通透、低饱和，营造清凉、安静、梦幻的 luminous mood。
- 可使用 blush pink、celadon green、ivory white、blue-gray shadows、deep jade green 等柔和倾向，但应根据原图主体和材质自然映射，不强行改变主体类型。
- 阴影保持轻薄、透明、偏蓝灰或冷灰，避免沉重黑影和强烈戏剧化对比。
- 光线应像柔和自然日光，画面保持清爽、干净、空气感充足。
- 禁止霓虹色、大面积高饱和背景、脏污灰暗色调和强烈金属反光。

## 背景与留白

- 背景保持纯白纸感、极浅色留白或非常轻的环境暗示，让主体像 sketchbook page 或 editorial illustration 中的手绘作品。
- 若原图背景对理解主体、空间关系或场景身份很重要，应保留其必要结构，并以淡化、松散、低对比的方式重绘。
- 不要扩写复杂新场景，不要加入与原图无关的新主体或装饰元素。
- 尊重原图画幅、主体完整性和主要构图关系，不强行改成固定海报版式。

## 主体保留

- 人物、动物、物体、建筑、食物、植物和自然场景都应保留原有身份与关键特征，只迁移绘画语言和明亮留白气质。
- 若原图不是花卉或植物，不要添加花、叶、池塘、种荚等特定植物元素。
- 可简化琐碎纹理和照片级细节，但不能丢失主体的轮廓、姿态、比例、材质线索和重要道具关系。
- 画面应具有 botanical sketchbook 之外更通用的 editorial hand-painted quality，干净、冷静、轻盈而完整。

## 禁忌

禁止添加文字、水印、Logo、边框、界面元素、乱码、无关装饰、新人物、新道具、固定植物意象、海报排版、标题栏、说明段落、摄影棚写真、过度锐化、脏污背景、强烈渐变、赛博朋克、低质卡通化和与原图内容冲突的新场景。
