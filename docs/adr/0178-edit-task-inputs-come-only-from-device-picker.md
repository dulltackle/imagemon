# 编辑任务输入只来自设备相册或文件选择器

手机应用首版的编辑任务，其 `image` 和 `mask` 文件输入只能通过系统相册或文件选择器从设备选取，选取后按 0016 作为任务历史的内部附件保存。首版不提供"把图片结果直接作为编辑输入"的入口，也不在编辑表单中浏览或引用图片结果。

这样可以避免在图片结果与编辑输入之间引入新的引用关系，与 0015（任务历史弱引用图片结果）、0016（不建立独立任务素材资产）、0048（图片结果不去重）保持一致。使用者如确需复用某张图片结果，可先按 0042 将其导出为相册中的非托管副本，再从相册选取，由此形成的"生成→编辑"闭环是间接的、经过用户显式导出的，而不是应用内的直接引用。

后续若要支持应用内直接选取图片结果作为编辑输入，需要重新评估引用语义、缺失资产处理和备份边界，并另行记录决策。
