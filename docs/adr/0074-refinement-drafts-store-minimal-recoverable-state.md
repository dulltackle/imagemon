# 提炼草稿只保存可恢复流程的最小状态

提炼草稿只保存恢复当前模板提炼流程所需的最小状态，包括使用者提供的外部完整提示词、计划用途、当前提炼方案结构化结果、已标注的提炼补充、草稿状态，以及非敏感文本模型配置快照中的配置名称和模型名。当草稿已含提炼方案时，「当前提炼方案结构化结果」必须包含将写入个人图鉴条目的逐字正文和输入声明，使恢复草稿后的本地确认写入（0184）能按 0195 逐字展示并批准、无需重新生成方案；否则本地确认写入将缺少可批准的正文而被迫退化为模型调用，违反 0184/0030。提炼草稿不保存原始完整模型响应、请求头、API Key、底层堆栈、中间推理内容或其他不可展示调试信息。
