# 恢复时重映射所有指向图片结果的引用，含封面

ZIP 恢复时，若图片结果按 0069 被分配新的内部 ID，应用统一跟踪一份"备份内原 ID → 本次恢复新 ID"的映射，并把备份内所有按内部 ID 指向图片结果的引用都重写到新 ID，包括任务历史对图片结果的弱引用（0015）和个人图鉴条目的封面示例引用（0045）。这样跨设备恢复后，任务历史的历史解释关系和条目封面链接都不会因 ID 重分配而断裂。

0079 在描述"保持弱引用和历史解释关系完整"时只列举了任务历史、图片结果和内部附件，未显式提到封面示例引用；本决策把同一重映射规则明确扩展到封面示例引用，与 0045（完整 ZIP 备份迁移封面引用所指向的图片结果）一致，避免出现"任务历史链接保留、条目封面却莫名变空"的不一致。封面引用的目标图片结果在备份中缺失时，仍按 0045 恢复为空封面；本决策只针对目标存在但 ID 被重分配的情形。
