# 仅通过用户主动导出的 ZIP 备份迁移数据

本地优先应用默认排除 iCloud 与 Android Auto Backup，不将 SQLite、图片、素材或 API Key 交由操作系统自动云备份；跨设备迁移只能通过使用者主动导出的明文 ZIP 备份完成。备份包包含格式版本、资产清单、哈希、个人图鉴条目、任务历史、任务快照、图片结果、非敏感模型配置、应用默认规格和任务历史内部附件，恢复前完整校验并合并导入，不直接覆盖应用目录。API Key 和提炼草稿始终不进入备份；如果某条任务历史本来就记录附件缺失，恢复时允许保持缺失状态。

导出备份前，应用展示备份包含内容和不包含内容，并由使用者确认后生成明文 ZIP；应用不做自动敏感信息扫描，也不加密备份文件。

备份 manifest 包含 `encryption: "none"` 字段。首版导出只生成明文 ZIP，恢复只接受 `none`；遇到其他加密值时提示当前版本不支持。首版不提供密码、密钥派生或加密算法选择。

恢复导入非敏感模型配置时，不导入 API Key，不覆盖当前设备已有模型配置或默认配置；导入后的模型配置处于未就绪状态，使用者补充 API Key 后才可设为默认并用于模型调用。名称冲突时创建导入副本名称，避免恢复流程因配置命名中断。

恢复导入任务历史时，不按内容去重、覆盖或合并现有任务历史；应用为导入内容生成当前设备的新记录 ID，并保留原始发生时间、任务状态和任务快照。图片结果同样导入为新的图片结果或新文件标识，恢复报告显示导入数量，重复内容由使用者后续手动清理。

恢复导入应用默认规格时，不自动覆盖当前设备的应用默认规格；恢复预览显示备份值和当前值，由使用者选择是否采用备份值。任务历史中的执行规格随任务快照恢复，不受应用默认规格合并选择影响。
