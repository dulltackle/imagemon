# 图片规格能力兼容校验只针对已知模型

0039 的"图片规格与模型能力是否兼容"校验只适用于已知模型，即应用按 0068 自带能力知识的模型。对这些模型，规格与模型能力不兼容时按 0039 禁用执行、要求使用者修改规格或切换模型配置，应用不自动降级。对未知模型，应用没有可靠能力表，不做本地能力兼容拦截，图片规格随调用透传给兼容平台；服务端若不支持该规格，则按失败任务处理并保存 0025 错误摘要。

这把 0039 与 0068（已知模型本地能力校验、未知模型透传）统一起来，并与 0183（输入文件格式/大小交服务端裁决）采用同一思路：应用只在确有能力知识时本地拦截，否则交由服务端权威裁决，避免对自定义 OpenAI 兼容服务（0032）误拦其实际支持的规格。

与模型能力无关的应用级规格合法性（例如数量至少为 1、格式取值为应用识别的有效值）不属于本决策，始终校验，与所选模型是否已知无关。
