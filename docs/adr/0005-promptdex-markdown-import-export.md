# 使用 Promptdex Markdown 导入导出个人图鉴条目

个人图鉴条目以 Promptdex Markdown 契约导出，文件包含输入声明、模板正文和条目展示信息。符合当前契约的文件经本地校验后可直接导入个人图鉴；普通外部完整提示词仍须经过模板提炼与确认，无效、展示信息缺失或版本不兼容的 Markdown 则拒绝导入且不自动修复、补齐或升级。

导入单个图鉴条目导出时，如果待导入名称与当前提示词图鉴中的内置图鉴条目或个人图鉴条目冲突，应用要求使用者选择修改待导入名称或跳过该条目。应用不覆盖、不合并现有条目，也不自动生成新名称；修改名称只发生在导入写入前，并可在同一步可选调整展示标题以便区分。导入冲突处理不允许修改模板正文、输入声明或用途说明等执行语义相关内容，写入后的个人图鉴条目仍保持不可变。
