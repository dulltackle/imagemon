# 手机端首版包含经确认的模板提炼

手机应用首版支持从一个外部完整提示词和明确计划用途中提炼新的个人图鉴条目，但必须先生成提炼方案并获得使用者确认后才写入。提炼方案展示拟写入名称和展示信息，使用者可在确认前修改名称和展示信息；最终名称必须通过契约校验且不与当前提示词图鉴冲突。提炼补充需要明确展示并获得批准，写入前必须通过本地图鉴条目契约校验；手机端复用 Builder 的提炼策略和领域边界，但不运行 Builder skill，也不支持一次融合多个完整提示词或修改已有图鉴条目。

提炼模型可以在方案中建议标题、用途说明、图鉴分类和搜索标签等展示信息，使用者确认前可编辑；这些展示信息建议不属于提炼补充。只有进入模板正文或输入声明、会影响执行语义的新规则，才作为提炼补充明确标注并获得批准。
