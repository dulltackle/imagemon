# 个人图鉴条目初始存储只包含 Promptdex 契约字段

个人图鉴条目的持久化以结构化字段作为真相源，而不是把整份 Promptdex Markdown 作为不透明文本保存；这样名称唯一性、合并图鉴读取、任务快照和后续导出都可以直接依赖本地字段。为了先打通个人图鉴条目与模板提炼写入，本次竖切的表结构只包含 Promptdex Markdown 契约中的名称、描述、版本、输入声明和模板正文，暂不存储标题、用途说明、图鉴分类、搜索标签或封面引用等手机端条目展示信息。这个收敛不改变既有产品决策：展示信息仍属于个人图鉴条目模型，后续实现时通过 schema migration 补入，并在补齐前用 `description` 承担列表和详情页的说明展示。
