# ZIP 导出期间禁止并发写操作

ZIP 备份导出在使用者确认导出范围后进入导出进行中状态。已有模型调用进行中时，应用可以允许使用者进入导出确认页查看将导出的内容，但必须禁用确认导出，直到当前模型调用结束。导出确认页停留期间必须实时跟随全局模型调用锁状态；只要图片任务、模板提炼、模型配置测试连接或拉取模型列表进行中，确认导出按钮就保持禁用并显示当前模型调用进行中的原因；模型调用结束后，应用应重新读取当前数据或重新校验确认页摘要是否仍有效，确认页内容仍有效时再自动恢复可确认状态。使用者确认导出时，应用必须按当前本地数据重新读取并生成备份，不能使用旧摘要或旧查询结果生成 ZIP。导出进行中期间，应用必须阻止图片任务、模板提炼、模型配置测试连接、拉取模型列表、图鉴编辑、单条导入导出、图片结果删除、任务历史删除、应用默认规格修改等会修改应用数据或触发模型调用的操作。导出进行中可以展示导出进度、取消导出和失败结果，但不能与其他写操作交错执行，以避免备份内容在读取过程中变得不一致。
