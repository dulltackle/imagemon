# 内置图鉴条目允许复制 Promptdex Markdown

手机应用允许设备使用者在内置图鉴条目详情页将该条目的 Promptdex Markdown 文本复制到剪贴板，但仍不提供内置图鉴条目的单条文件导出。复制文本采用当前 Promptdex 契约可直接解析的模板 Markdown：frontmatter 包含 `name`、`description`、可选 `version` 和 `inputs`，正文保持原样，不额外加入手机端展示信息字段。复制是轻量的显式文本取用动作，不创建迁移文件、不携带封面示例图，也不改变 0044 中“单条导出只面向个人图鉴条目”的迁移边界；这样既能让设备使用者复用或检查内置条目的完整条目文本，又避免把内置条目纳入文件级导出、导入冲突和备份迁移语义。
