# 手机端采用适配式水彩视觉系统

在 ADR 0210 已固定浅色模式的前提下，手机端采用 Imagemon 自有的适配式水彩视觉系统：品牌表面可以使用克制的纸张底色与水彩 wash，输入、审阅、历史、模型配置和状态反馈保持近白实色工具表面，所有输入图片、图片结果、缩略图和图片缺失占位只落在独立的中性 media matte。这样既建立品牌识别，也避免界面配色影响设备使用者判断图片，并保持 iOS、Android 和 Web 的工具操作清楚可验证。

实现使用自有 `app-*` 语义 token、共享组件和独立生产资产；StyleKit 只作为设计参考，不是运行时依赖，原生 `NativeTabs` 与 `Stack` 继续使用平台组件。具体颜色、组件、资产、响应式和批准偏差以[移动端水彩视觉合同](../design/mobile-watercolor/design-contract.md)为准；若未来复制 StyleKit 的实质代码或资产，必须保留 MIT 归属。

本决策补充而不取代 ADR 0210，也不改变路由、信息架构、业务状态、任务执行逻辑或持久化。自定义字体、暗色、动态主题和设备使用者主题开关不在本决策范围；若未来改变冻结配色、恢复暗色或允许设备使用者切换主题，必须另立 ADR 并补齐对应的跨端视觉验证。
